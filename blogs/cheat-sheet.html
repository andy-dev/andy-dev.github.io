Cheat Sheet!

Ruby | JavaScript loops

Loop for each element in array

Ruby:

        array.each do |var|
            do something
        end

Javascript:

        for (var i = 0; i <= limit; i++) {
            do something;
        };

Loop while a condition is true

Ruby:

        counter=1
            while counter != some condition
                do something
                counter = counter +1
            end
        end

Javascript:

        var i = 10;
        while (i >= 0) {
            do something;
            i--;
        };

Loop until a condition is true

Ruby:

        until x == y
            do something
        end

Javascript:

        while (x != y) {
            do something;
            i--;
        };

Do first, then loop while a condition is true

Ruby:


        begin
            puts n
            n -= 1
        end while n > 0

Javascript:

        var i = 10;
        do {
            do something
        } while (i >= 0);

Loop through a collection

Ruby:

        for i in 0..n
            do something with "#{i}"
        end

JavaScript:

        for (attribute in object) {
            do something with object[attribute];
        };
